    Block Coding!

    What exactly is block coding? Block coding is a method used in computer science and information theory to represent and transmit data in a structured and organized manner. It involves dividing data into fixed-size blocks or groups of bits, each containing a predefined number of binary digits (0s and 1s). These blocks are created with specific patterns and redundancy information, which helps in error detection and correction during data transmission or storage.

    A text-based programming language is a type of programming language where instructions are written using plain text. In contrast to visual or block-based programming, which involves dragging and dropping visual elements, text-based programming requires users to write code using a specific syntax and structure. Each programming language has its own set of rules and keywords, and developers use a text editor or an integrated development environment (IDE) to create, edit, and save their code.

    Coding Structures!

    Variables!

    Variables serve as containers for storing and manipulating data. When writing a program, you often need to work with different types of information.Variables can be used to represent values in a more dynamic way than hardcoding.

    Conditional Statements!

    A type of coding instruction used to compare values and express and make decisions. A conditional statement tells a program to execute an action depending on whether a condition is true or false. It is often represented as an if-then or if-then-else statement.

    Functions!

    Functions are "self contained" modules of code that accomplish a specific task. Functions usually "take in" data, process it, and "return" a result. Once a function is written, it can be used over and over again. Functions can be "called" from the inside of other functions.
